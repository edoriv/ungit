# Security Policy

## Supported scope

UNGIT is provided as-is.

Security-sensitive areas include:

- restore operations
- archive extraction
- file deletion/pruning
- MCP command execution

## Reporting

Please open a private report if possible, or open a GitHub issue with minimal exploit detail first.

Include:

- UNGIT version/commit
- macOS version
- steps to reproduce
- expected vs actual behavior
- impact assessment

## Notes

Do not post secrets, tokens, or private project data in public issues.
