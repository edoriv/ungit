# Support

UNGIT is free/open-source and currently maintained as available.

## Get help

- Open a GitHub issue for bugs or setup problems.
- Use Discussions (if enabled) for usage questions and workflow ideas.

## What to include in support requests

- macOS version
- UNGIT version/commit
- whether you run UNGIT from Xcode or `/Applications`
- MCP setup values (command path + working directory)
- logs/screenshots and exact command used
