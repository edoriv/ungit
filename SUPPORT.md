# Support

UNGIT is free/open-source and currently maintained as available.

## Maintenance expectations

UNGIT is primarily maintained for personal use.

- I may keep improving it when I need to.
- I may fix bugs that affect my own workflow.
- I may review issues or pull requests when time allows.
- I am not treating this as a request-driven product.
- I am not promising feature work, roadmap work, or response times.

If the project is useful to you, that is great.
If you want to build on it, fork it, or improve it for your own needs, you are welcome to do that under the license.

## Get help

- Open a GitHub issue for bugs or setup problems.
- Use Discussions (if enabled) for usage questions and workflow ideas.

## What to include in support requests

- macOS version
- UNGIT version/commit
- whether you run UNGIT from Xcode or `/Applications`
- MCP setup values (command path + working directory)
- logs/screenshots and exact command used
